LinkMe Node SDK

Server-side link management, webhook verification, and backend automation. Create, update, and manage deep links programmatically.

SDK Reference npm

1 Install

npm install @li-nk.me/node-sdk

2 Create Links

import LinkMeClient from '@li-nk.me/node-sdk';

const linkme = new LinkMeClient({
    apiKey: process.env.LINKME_SERVER_KEY,
});

const link = await linkme.createLink({
    appId: 'app_123',
    slug: 'spring',
    deepLink: '/promo/spring',
    redirects: {
        iosStoreUrl: 'https://apps.apple.com/...',
        androidStoreUrl: 'https://play.google.com/...',
        webFallbackUrl: 'https://example.com/spring',
    },
});

3 Verify Webhooks

import {
    verifyLinkMeWebhookSignature,
    parseLinkMeWebhookEnvelope,
} from '@li-nk.me/node-sdk';

const rawBody = req.rawBody.toString('utf8');
const signature = req.get('X-LinkMe-Signature');

if (!verifyLinkMeWebhookSignature(
    rawBody, signature, process.env.LINKME_WEBHOOK_SECRET!
)) {
    res.status(401).send('Invalid signature');
    return;
}

const envelope = parseLinkMeWebhookEnvelope(JSON.parse(rawBody));

Features

Link Management

Create, read, update, and delete deep links programmatically. Automate campaigns and marketing flows.

Webhook Helpers

HMAC signature verification and typed envelope parsing for secure, reliable event processing.

Zod Schemas

Runtime validation with exported Zod schemas and inferred TypeScript types for all API payloads.

ESM & CJS

Dual-publish support. Works with import (ESM) and require (CommonJS) out of the box.
